# Лабораторная работа 4 (Связана с презой 12)

![alt text](4.png)

## Текстовое описание:

1. Сгенерировать модельный ряд xx = norm(0,1), k = norm(0,1), k = 1, ..., 195; 
   - Дополнить его пятью значениями: 5, -4, 3.3, 2.99, -3. 
   - Отсортировать ряд и применить правило трех сигм для проверки на аномальность для трех первых и трех последних порядковых статистик.
2. Построить для сгенерированного ряда боксплот Тьюки, выделить обнаруженные аномальные значения. 
   - Сравнить с результатами предыдущего задания. Сделать выводы.

## Выводы

- Правило трёх сигм, применённое только к крайним значениям, выявит меньше аномалий, 
  если не все из них выходят за пределы трёх стандартных отклонений
- Боксплот Тьюки использует межквартильный размах (усы = 1.5*IQR), что более устойчиво
  к ненормальным распределениям и выделяет выбросы по квартилям.
- В данном случае добавленные значения 5, -4, -3, являются выбросами в обеих методиках
- Боксплот может выявить больше или другие выбросы при наличии скошенности данных.
