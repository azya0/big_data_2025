1. Чем отличаются друг от друга постановки следующих задач машинного обучения: обучение по прецедентам, обучение с учителем, обучение без учителя?

Обучение по прецендентам - буквально общая пародигма машинного обучения. Самая абстрактная из вышеописанных задач, в рамках которой из прецендентов нужно найти закономерности.

Обучение с учителем - обучение по данным вида (вход, правильный выход), в рамках которого мы обучаем модель находить закономерности по заранее размеченным данным.

Обучение без учителя - подход, при котором мы имеем только входные данные, из которых нужно вычленять закономерности для получения правильного ответа.

2. Приведите примеры ответов (откликов), которые можно отнести к булевым, номинальным, порядковым и количественным. Свяжите тип отклика с видом функции потерь.

Булевый - одно число от 0 до 1, которое мы округлим. Связь с функцией потерь: Бинарная кросс-энтропия (log loss). Она штрафует не только за неверный ответ, но и за низкую «уверенность» модели в правильном классе.

Номинальный (категориальный) отклик - n конечных чисел (для n классов), выберем наибольшее. Связь с функцией потерь: Категориальная кросс-энтропия. Это прямое обобщение бинарной на случай многих классов, не связанных порядком.

Порядковый отклик - число от 0 до n, (где n - количество классов).
Связь с функцией потерь: взвешеннач версия кросс-энтропии, с усилением штрафов тем больше, чем дальше от нужного класса.

Количественный отклик - некоторое число без явных границ. Связь с функцией потерь: Среднеквадратическая ошибка (MSE) или Средняя абсолютная ошибка (MAE).

3. Другие примеры функций потерь

Один из моих любимых подходов, применяющийся для обучения GAN моделей - Adversarial Loss. Суть с том, что по мимо основной модели, мы пишем дополнительную, оценивающую реалистичность результатов. Из оценок этой модели и берётся потеря.

4. Имеется заданное множество людей с известными значениями их роста и охвата груди (или веса). Ставится задача разработать небольшое количество типовых размеров одежды для них. К какому типу задач машинного обучения относится сформулированная? Опишите формальную (математическую) постановку этой задачи

Дано:

```
X = {x_1, x_2, ..., x_n}, где n - количество людей

x_i = (height, weight) - двумерный вектор
```

Найти:

```
1. Разбиение множества X на k непересекающихся подмножеств
2. Для каждого разбиения определить центроид (типовой размер)
```

Это каноническая задача кластеризации в рамках обучения без учителя.
